# ADR-004: 결속력 분산도(보유량 균등도) MVP 제외

## 상태
보류 (2단계 이후 검토)

## 배경
서비스의 핵심 목표는 "결속력"이다. 결속력을 완전히 수치화하려면
참여자수와 보유금액 외에 **보유량이 얼마나 고르게 분산됐는지**도 봐야 한다.

예시:
```
왕국 A: 1명이 9,900주 + 99명이 각 1주  → 사실상 개인 독점
왕국 B: 100명이 각 100주               → 진짜 결속력

현재 공식으로 A와 B의 power가 동일하게 나온다.
```

분산도를 반영하는 방법으로 표준편차나 HHI(허핀달-허시만 지수)를 검토했다.

## 결정
MVP에서는 분산도를 공식에 포함하지 않는다.

## 이유
- 집계 쿼리에서 표준편차/HHI 계산은 복잡도가 크게 올라간다.
- `log(참여자수 + 1)` 항이 소수 몰빵을 약하게나마 억제한다.
  (1명→2명은 log값이 크게 뛰지만, 100명→101명은 거의 안 뜀)
- 완벽하지 않지만 방향은 맞고, MVP에서 데이터를 쌓은 뒤
  실제 왜곡이 얼마나 발생하는지 확인하고 개선하는 것이 현실적이다.

## 다음 검토 시점
실제 유저 데이터가 쌓인 후, 보유량 편중이 심한 왕국이 상위권을 독점하는
현상이 관측되면 분산도 항을 추가한다.
